# 卜蜂莲花客户端的 React-native版
目前仅基于iOS版进行开发，可能还不适配安卓端

# 本程序相关接口都是基于原程序进行抓包完成
对源程序的仿真度几乎百分百

## How to run

1. Prepare your environment
2. Clone this repo, and goto the project root directory
3. run `npm install --save`
4. run `react-native run-ios`
5. Enjoy
